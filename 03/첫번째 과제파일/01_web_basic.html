웹 접근성

장애 유 무를 떠나서 모든 사용자가 웹에 접근하여 동일한 기능 동일한 컨텐츠를 사용할 수 있어야 한다.(마우스, 키보드, 음성브라우저)

장점: 누구나 접근하여 같은 기능, 같은 컨텐츠를 제공받을 수 있음!
단점: 디자인이나 기능을 제대로 활용하기 어려움
+ 비디오, 애니메이션 사용에 제한이 있음 (초당 3~50회 깜박이는 증상이 없어야 함!)
+ 비디오 자막제공 or 수화제공
+ 사이트의 목적이 상실 될 수 있음(기능, 디자인에 제약이 많음)
+ 벌금 (3000만원 이상)

웹 표준

모든 브라우저들은 제공하는 기본 값이 다르기 때문에 일정한 규칙을 만들어서 "같은 결과물" 이 나오도록 만들어야 한다! (법률제한)

글꼴, 색상, 폰트크기, 비디오 플레이 재생 위치가 동일하게 출력되어야 하고, 애니메이션 기능 등은 컨트롤 할 수 있어야 한다! (웹 표준)

웹 표준을 따르는 브라우저들

표준브라우저: 크롬, 오페라, 파이어폭스, 사파리, 익스플로러 9, "10, 11"
9버전은 표준 브라우저에서 사용할 수 있는 효과나 기능을 활용하기 어려움!
10버전 부터 안전하게 사용이 가능함! (가장 안전한 버전 익스플로러 11)
