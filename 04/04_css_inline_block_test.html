인라인요소란? em,strong, span, i, b, img 처럼 특징이 있는 태그들도 많고, 글씨 요소처럼 옆으로 나열 할 수 있다. 너비와 높이를 가질 수 없음! ( 담고 있는 글씨 크기, 객체 크기 만큼만 영역을 차지함!)

인라인요소와 블록요소 구분하기

1) 해당하는 태그 복사하여 붙여넣기
2) 옆으로 나열되면 인라인요소, 밑으로 나열되면 블록요소
3) 배경색상으로 확인가능
+ 영역을 끝까지 차지하면 블록요소
+ 영역을 타이트하게 차지하면 인라인요소
