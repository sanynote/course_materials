+ xampp 설치하기

01) 아이콘 2번 클릭
02) 알림창 / ok
03) next 버튼 클릭
04) ampp 순으로 설치하기 (아래부분 4개 체크)

    + Apache (모든 서버 연결시 매우 중요)
    + Mysql (cmd창에서 사용되는 언어(문법))
    + PHP
    + phpMyAdmin

05) next 버튼 클릭 (설치할때까지 클릭) learn more~어쩌구 체크박스 해제
06) 중간에 브라우저 화면 보이면 꺼도 상관없음

07) 설치중에 방화벽 엑세스 허용 클릭
08) 설치 종료 후 Finish 클릭
09) 만약 국기선택 나오면 "미국국기선택"

설치 후

10) panel에서
    Apache와 Mysql 옆에 start 버튼 클릭

매우중요) 
    80포트, 3306포트 연결되는지 확인
