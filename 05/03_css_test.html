01) 표현 영역이란? 어느 영역인지 작성하기

표현영역이란 style 영역을 뜻합니다.

02) 구조 영역이 없을 경우 생길 수 있는 문제는?

철근이라도 있어야 유리나 벽이라도 칠하듯이, 표현이나 기능을 만들어봤자 구조영역이 없으면 아무것도 보이지 않는다.

03) style 적용효과 3가지 방법 나열하고, 각각 어떤 식으로 작성하는지 내용 작성하기

1) inline-style 직접표현방식 : 해당 태그 내에 직접 표현을 적용시킨다. 2) internal-style 내부표현방식 : 내부 head에 style로 표현을 적용시킨다. 3) external-style 외부표현방식 : 외부에 style을 적용시킨다.

04) style적용방식 중 태그에 직접 적용하는 방식은?

inline-style(직접표현방식) 입니다.

05) 기능영역에 해당하는 2가지 영역은?

둘 중 조금 더 빠른 javascript 와 jQuery 가 있다.

06) 05번에 해당하는 값 중, 가장 느리게 읽어주는 영역은? 그 이유는?

17) 어떤 태그에 text-align을 적용하였다. 적용되지 않았다면 이유는?

그 태그가 인라인요소라서 적용되지 않았을 것이다.
인라인요소는 너비와 태그, 여백을 가지고있지 않아서 글자배열이 적용되지 않는다.

19) img태그에서 대체 텍스트로 사용하는 속성이고, 해당하는 값이 필요한 이유는?

alt속성이 대체 텍스트로 사용됩니다. src만 보고는 이미지에 대한 설명과 이해도가 떨어질 경우 보충설명을 할 수 있습니다.

20) 인라인요소를 블록요소에 담아주는 이유는?

인라인요소는 너비와 높이를 가지지 않기때문에 글자배열을 정렬할 수 없습니다.
그러한 경우 블록요소에 담겨있다면,부모요소인 블록요소의 style에 배열을 적용하면 글자를 정렬할 수 있습니다.

22) 슬라이딩 윈도우나 광고가 적용되는 영역명은?
