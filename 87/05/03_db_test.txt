2) 한글 키보드 입력될 수 있도록 셋팅하기

[매우중요]

	shell (cmd)창을 열고 한글 셋팅을하면 열려있는동안만 셋팅이 가능함
	: 창을 닫고, 다시 열었을때는 다시 셋팅해야 한글키보드입력이 가능람

//////////////////////////////////////////////////

set character set euckr;
SET CHARACTER SET EUCKR;

//////////////////////////////////////////////////
