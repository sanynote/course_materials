html5에서 배우는 태그와 효과(필수)

  • 시멘틱 태그 (header(nav), section, footer(address))
  • transform (변형)
  • * 부드럽게 바뀌는 효과 (php까지 이어질 수 있음)
  • * form (type들), 로그인과 회원가입 페이지
  • animation, video (움직이는 효과들)
  • 반응형, 절대형, 적응형 (html5와 script 사이의 한 과목)

transform: html5에서 객체들을 "변형" 할 때 사용하는 효과

  • 1) text-transform : 글자 변형할 때 사용하는 효과
  • 2) 소문자 변형하기 : lowercase
  • 3) 대문자 변형하기 : uppercase
