display:none과 display:block

display:none 화면에서 필요한 영역만 남기고 방해되는 부분을 숨길 수 있음
오류가 나거나 에러가 발생했을 때 화면에서 숨겨놓을 수 있음.

1) display:block, 인라인요소를 block으로 만들 때 사용
2) display:block, display:none으로 사라진 객체를 화면에 다시 나타나게 할 때 사용

매우중요) 숨겨진 객체는 화면에 없음, 마우스를 올릴 수 없음

정리) 객체를 만들때, 부모 자녀를 만들어주는 이유는? 자녀가 사라지면 부모에서 해결이 가능함

margin-top, float, position 화면에서 사라지는 객체 등등 , 모두 부모요소에서 해결이 가능함

display:none 사용시 주의사항

headline 태그 사용이유? html5 에서 목차생성을 위한 필수태그

* 해당 페이지를 검색할때도 목차와 meta 태그로 접근이 가능함

display:none은 화면에서 완전히 사라짐

h1~h6 까지 사용하는 headline 태그에서 display:none을 사용하면
목차생성은 물론 해당 페이지를 인지하지 못할 수 있음

strong, em (스크린 리더기가 읽을 수 있는 태그)
display:none으로 숨기고, display:block으로 다시 나타나게 할 경우
숨겨졌을때는 읽지 못하다가 나타날때 다시 읽음

display:none은 많이 사용하는 효과로 꼭 필요한 영역에서 사용중인지
인지가 가능해야하는 영역에 무분별하게 사용중이지 않은지 확인하면서 작업하기!
