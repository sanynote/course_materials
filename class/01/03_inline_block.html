웹의 가장 기본적인 요소

웹에서 태그를 만들면 인라인이나 블록속성 둘 중에 하나인 속성을 가지게된다.

블록요소

블록요소란? 기본적으로 너비와 높이를 가지는 박스형태의 요소

블록요소는 처음부터 만들어때 영역을 끝에서 끝까지 전부 차지함
옆으로 나열될 수 없음
중요) 블록요소는 기본적으로 아래로 떨어지는 성질을 가지고있음

인라인요소

인라인요소란? 글씨처럼 옆으로 나열되는 형태를 인라인 요소라 말한다.

인라인 요소들 중에는 한 줄로 처리되는 br태그같은 태그도 있지만
span 태그처럼 글씨를 묶어주는 열고 닫고 태그도 존재한다.
글씨를 담아주는 특징이 없는 태그
인라인요소는 가나다라마바사... 처럼 옆으로 나열되는 특징을 가지고있음
br태그처럼 한 줄로 끝나는 태그에는 가장 뒤에 슬러시 표현을 해서 닫아주어야한다.

br태그 제대로 사용하기

블록요소는 기본적으로 위아래로 여백을 만듦
인라인요소는 글씨나 글자 요소만큼 여백을 차지함


블록요소 뒤에는 어떠한 경우에도 br태그를 사용하지 않음
: 좌우,위아래 공간을 만들때는 css 효과로 만들 수 있음 블록요소 뒤에 br을 사용할 경우, script에서 정밀한 작업을 할 때 방해가 되기 때문에 문제가 커지는 부분
제대로 사용하기 위해선, 위에서 아래 글자 떨어뜨릴대 사용하거나
인라인요소가 끝난 부분 뒤에다 사용하여 다음 인라인요소를 떨어뜨릴때 사용한다.

br태그 확인하기

1) 블록요소 뒤에는 절대 br태그를 사용하지 않는다!
2) 인라인 요소에서 enter를 사용하여 줄바꿈할때 enter 역할을 하는것이 br태그
3) 태그 안에서 더 이상 떨어뜨릴게 없다면 br태그 제외하기
4) 태그와 태그를 떨어뜨릴때 태그가 닫힌 부분 뒤에 br태그 사용하기
5) 현재 위의 h3태그와 그 위의 span 태그 사이에는 br태그 없음!
블록요소는 알아서 아래로 떨어지기 때문에 br을 구분하지 않아도 상관없음
