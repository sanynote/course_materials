웹을 읽는 순서

  1. body영역(구조물 작성, 기본적인 기초작업)
  2. style영역 (기초작업에 꾸며주는 작업, body 없으면 style 영역은 꾸미기 어려움)
  3. script영역 (javascript, jQuery)
  4. javascript (jQuery보다 빠름)
  5. jQuery(javascript + query의 약자표현)
  6. db 작업(mysql, php 작업)
