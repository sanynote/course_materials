selected 태그 (선택하는 태그)
select 태그는 img처럼 많이 사용되는 인라인 요소들 중에 하나
회원가입이나 어떤 선택하는 용도로 사용되는 대표적인 태그들 중 하나
기능을 위해 많이 사용되는 인라인요소: img, select, a
중요속성이 많기때문에 반드시 블록요소에 담아서 사용해야한다.
많이 사용되는 곳: 언어선택, 과목선택, 항공사 지역선택, 패밀리사이트(제휴), 회원가입(이메일 선택)
* select 태그 사용시 필수로 사용되는 태그는 option태그
자녀요소가 아니라 select가 선택하는 option
select option (선택불가)
option은 묵음처리되는 태그
option에서 중요한것은 "value"속성
select 태그에 이름 지정하기(id 주로 사용)
select 태그처럼 대표적인 인라인요소들은 부모요소에 담아서 사용해야하는 경우가 많음
부모를 통해서 #footer select 형식으로 작성되기보다 직접 #lang, #type 선택되는 경우가 많음
필수 작성, select 태그에 id값 || class 필수로 작성
javascript에서 태그에 직접 접근하여 option value값을 선택해야 하는 기능이 많음
option 태그에 사용할 수 있는 속성으로는 value와 selected가 있음
selected는 해당하는 option이 미리 선택되어있도록 만드는 속성
