jQuery를 사용하는 이유는? javascript "조금 편리" 하게 사용하기 위한 목적을 가짐

jQuery 1 단계 : style영역 사용하는 css와 + body의 태그 + jQuery 함께 사용하기

* jQuery 2단계 : 태그(인라인과 블록요소) + style + jQuery + javascript (제어문, 반복문, 배열...)
