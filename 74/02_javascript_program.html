javascript 와 java의 차이점

java의 특징

  1. java는 컴파일이 되어진 소스만 읽는다.
  2. 컴파일이란? 컴퓨터가 알아들을 수 있도록 번역 (해석) 해주는 작업
  3. 컴파일을 하기위하여 사용하는 프로그램 : 이클립스
  4. 웹 표준, 웹 접근성과 전혀 관련이 없음, 태그롸 css 부분을 다루는 영역이 없음
  5. 브라우저를 고려하지 않아도 전혀 문제가 되지 않음
  6. 소스공개가 필요없음
  7. * 기본적으로 웹과 관련이 없음

javascript의 특징

  1. javascript는 컴파일이 되어있지 않아도, 브라우저에서 바로 읽을 수 있음
  2. javascript는 브라우저에서 제공되는 모든 기능을 구현할 수 있음
  3. javascript로 기능구현시, 웹 표준, 웹 접근성을 고려할 수 있음
  4. 반드시 브라우저에서 사용이 가능해야 함
  5. body 영역, style 영역 모두 컨트롤이 가능해야 함
  6. 웹 표준과 웹 접근성을 고려하기 때문에 소스 공개를 원칙으로 함
  7. 정리) 웹에서 기능과 꾸며주는 작업은 javascript로 진행하고, 웹 관련이 없지만 프로그래밍을 하는것은 java로 한다.
