html5에서 사용하는 form

form에 대한 중요 속성들

  1. action: 주소를 연결하기 위한 중요 속성(control페이지나 database로 넘겨줄 수 있음)
  2. method: 주소줄에 넘어가는 정보를 보여줄지, 보여주지 않을지 결정할 때 사용
  3. * get : 주소줄에 넘어가는 정보를 보여줘도 될 때 사용(검색이 가능한 페이지에서 사용)
  4. get을 사용하면 몇가지의 정보를 추가로 더 가져올 수 있음
  5. * post : 주소줄에 넘어가는 정보를 숨겨야할 때 사용 (로그인,회원가입,중소기업(페이 내 검색))
  6. get,post,(request)

html5에서 type 사용에 대한 목적

  • type을 사용하는 이유: "사용 목적에 맞춰서" 작성하기 위하여 필수 적용
  • [매우중요] : html5는 각종 디바이스들에서 화면에 출력되기위하여 제작이 가능함
    : screen + 인터넷 지원 가능한 기기들
  • html5, 스마트폰 (세로,가로), 태블릿pc, 일반pc
    [매우중요] : 키패드 모양이 적절하게 출력되도록 만듦
  • type을 제대로 적용받기위해서는 "표준 브라우저"를 사용해야 함
  • * type (validator에서 에러가 발생할 수 있음)
    * 기술력은 앞서있지만, 브라우저가 따라오지 못하는 문제(크롬,사파리)
  • 익스플로러는 다른 표준 브라우저와 다르게 "표준 브라우저" 라고 해도 지원되지 않는것들이 많음
  • 익스플로러, 몇 버전부터 표준? 익스플로러 9버전부터 표준이지만, 10버전부터 지원가능한 게 많음
  • 10버전에서 type 지원이 어려운 것들이 많음
  • 가장 안전한 익스플로러는 11버전
  • * type 지원이 어려운 브라우저라면 script에서 제작하여 만들기
