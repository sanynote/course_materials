select태그와 video 태그
* select 태그의 option태그는 선택이 가능할까? option 태그는 선택이 불가능하지만 value 속성은 선택 가능
* video 태그의 source태그는 선택이 불가능함, src, type만 선택이 가능함
script에서 선택하는 방법 : select.value / video.src

video 활용하기

  1. video태그는 "인라인요소", input, img, select처럼 스스로 너비와 높이를 가지고있음
  2. video 태그에 width="640px" height="480px", validator 확인 시 에러 발생
  3. style 적용이 방식이 아니라 사용자가 직접 너비와 높이를 지정하여 작업함
  4. 위와 같은 방식은 브라우저에서 이미 px로 인식하기때문에 오류가 발생함
  5. 브라우저가 제일 좋아하는 단위는 px
  6. 640pxpx, 480pxpx 단위가 이미 작성되어있어서 충돌이 발생함
  7. 제일 안전한 방식은 style에서 수정하는 작업이 가장 안전하다
  8. mp4, ogg, webM 이외의 확장자는 재생불가능

mp4 확장자가 재생되지 않는 이유

mp4는 모든 브라우저에서 지원이 가능함, 익스플로러는 10버전부터 재생이 가능함 : 모바일용과 pc용으로 구분했을때 모바일용이 재생불가능한 경우가 있음 : 모바일용으로 인코딩을 새로 해주어야 재생이 가능하다
비디오가 재생되지 않을때, 대체텍스트와 마찬가지로 poster 속성을 사용하여 이미지로 대체함 애플광고의 경우, 애플 로고나 애플 관련 이미지로 대체
