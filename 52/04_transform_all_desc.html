transform: "변형"할 때 사용하는 효과

"transform"은 변형 시 모든 축이 가운데 있음

"transform"이 적용되지 않은 객체들은 "왼쪽상단" 을 기준으로 움직임

이동 "변형" : translate(기본단위 px)

이동 x and y : transform:translate(x축 y축)

이동 x 축 : transform:translatex(x축으로 이동)

이동 y 축 : transform:translatey(y축으로 이동)

transform 사용 시 중요사항

  • transform은 "모든 축이 가운데" 있음, 기능이나 반응형 작업 시 "초기값" 필수
  • style의 경우, 스스로 초기값을 인식할 수 있음
  • script의 경우, 스스로 초기값을 인식할 수 없어서, 사용자가 직접 만들어야 함
  • 기본 크기: font-size:16px; font-weight:400
  • * transform:translate, position:fixed는 자신의 위치값을 저장할 수 있음
  • * transform:translatex, translatey(scroll로 기능 만들때 중요한 효과)

크기 변형 : scale (100% == 1, 단위 없음)

크기변형 x and y transform:scale(x축 (1), y축 (1))

크기변형 x 축 : transform:scalex(0.5)

크기변형 y 축 : transform:scaley(0.7)

transform, scale 사용 시 주의사항

  • 초기값, 대다수 0,0으로 작업함
  • 100%를 기준으로 만드는 것이 scale, 100% == 1
  • scale은 기본적으로 100%를 기준으로 만들기때문에 초기값 1

회전 변형: rotate(45deg)

회전변형 2d(평면) : transform:rotate(45deg)

회전변형 x축 (3d) : transform:rotatex(60deg)

회전변형 y축 (3d) : transform:rotatey(80deg)

회전변형 3d축 : transform:rotate3d(x축 y축 z축 각도)

기울이기 : skew(x축 각도(deg), y축 각도(deg))

기울이기 변형: transform:skew(x축 각도, y축 각도)

기울이기 변형: transform:skewx(x축 각도)

기울이기 변형: transform:skewy(y축 각도)
