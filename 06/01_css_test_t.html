태그가 없다면 id 속성을 작성할 값이 없기 때문에 tag을 우선적으로 읽음!

style 영역, script 영역에서는 id 속성을 읽을때 태그보다 빠르게 읽음

1차 영역 body영역에서 tag와 id를 순서대로 읽음!

2차 영역 style에서 p를 호출하면 20~30개 이상을 호출해야 되기 때문에 느림

3차 영역 script 에서 p를 호출하면 body의 태그들 모두 읽고, style의 효과들 모두
읽은 다음 호출되기 때문에 매우 매우 느림

3차 영역 script에서도 id를 호출하면 중복되지 않기 때문에 빠르게 호출이 가능함!
