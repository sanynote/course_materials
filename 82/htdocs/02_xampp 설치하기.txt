1) 아이콘 2번 클릭
2) 알림창 / OK 클릭
3) Next 버튼 클릭
4) AMPP 순으로 설치하기 (아래 부분 4개 체크)
	
	+ Apache (모든 서버 연결시 매우 중요)
	+ MySQL (cmd 창에 사용되는 언어(문법))
	+ PHP
	+ phpMyAdmin

5) Next 버튼 클릭 (설치할 때 까지 클릭)
6) 중간에 브라우저 화면 보이면 꺼도 상관없음!

7) 설치 중에 방화벽 엑세스 허용! 클릭
8) 설치 종료 후 Finish 클릭
9) 만약에 국기 선택이 나오면 "미국국기" 선택!


10) panel에서 
	Apache와 MySQL 옆에 start 버튼 클릭
