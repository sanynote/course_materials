웹 설계시 해당하는 영역 사용하기
1) body 영역 필수 (웹의 구조를 작성하는 영역)
: body 영역이 없으면 style을 만들 수 없음
2) style 영역, body 영역을 꾸며주기위하여 작성
: 미디어쿼리: 반응형 페이지 만들기위하여 해상도 편집하는 작업
3) script 영역, 웹(브라우저)에서 사용되는 모든 기능들을 만들 수 있음
1) xhtml(정적인페이지,표준법) + script 기능 : script에 기능이 모두 작성되어있기때문에 "페이지가 흔들리거나 고장나는 경우" 가 많이 발생함
2) html5(정적인 페이지,동적인 기능(transition,animation)) + script 기능 : script에 의존하는 기능을 html5의 동적인 부분과 합쳐서 사용할 수 있음 : 기능의 속도가 빠르고, 안정성이 빠름
* 웹 분야에서는 어떤 분야도 js에 대한 이해도가 있는 경우를 선호함
(*) script의 "필수"기능
1) 아코디언 기능 (header, footer)
2) 탭 메뉴 (탭 베이직) 기능
3) 슬라이딩 윈도우 (visual에서 사용) : 슬라이딩 윈도우를 사용하는 이유는? 큰 장면을 이용할 수도 있지만 나열된 제품들을 하나씩 || 한 묶음씩 움직이게 할 수 있음
4) 모달팝업
5) js 버전 (판매가 * 수량), 계산식 페이지
6) google.com을 선택하면 옆에 input에 출력되는 페이지 만들기(도메인 선택시 input에 나타나는 기능)
7) js, 제어문(if) 반복문(for) 이용한 페이지 제작
8) 배열 (순차배열, 연관배열)
매우중요) js +jQuery + style + body 모두 섞어서 사용이 가능함
+ 반응형 슬라이딩 윈도우
