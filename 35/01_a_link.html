STEP01: 경로설정, 호출기호

  • 1) "#은 호출의 의미"로 사용하는 기호
  • #을 폴더명이나 파일명 앞에 기입하게되면 "다른 디렉토리"가 잘못 연결 될 수 있음
  • 2) 실행하는 파일이나 서버에 업로드하는 파일들은 다른 나라 언어 사용금지
  • 한국어,한자,일본어,불어 등등 사용이 어렵다. 주로 영문자(대문자,"소문자") 사용
  • php의 경우, 대문자와 소문자로 구분할 수 있음(예약어와 사용자가 기입한 언어)
  • 3)매우중요) 주소연결시 공백금지
  • href=" subpage.html"(X) 빈칸을 담고있는 서브페이지
  • href="subpage.html"(O) 경로 설정시 앞뒤로 빈칸 없이 사용하기

STEP02: 상대경로, 절대경로

상대경로: "현재 위치" 에서부터 "상대가 있는 경로를 찾아서" 링크를 걸어주는 작업

절대경로: 주로 기업에서 사용하는 경로, (주소가 변하는 경우가 거의 없음)

STEP03: 속성을 가지는 인라인요소 and 너비와 높이를 가지는 인라인요소

속성을 가지는 인라인요소

  • 1) img 태그, src와 alt, src는 이미지 파일을 연결하는 속성(javascript 접근가능한 속성)
  • src 와 alt 는 제어문(if)을 만들때 매우 중요한 속성에 속한다.
  • 2) select 태그, option(value), javascript 접근가능한 속성
  • 3) input(id(script),type(html5),name(php))
  • html5에서 사용가능한 type(로그인,회원가입 페이지 만들기)

* 처음부터 너비와 높이를 가지는 인라인요소(xhtml, html5, script에서 중요도가 높음)

  • 1) span strong em i b : 너비와 높이를 스스로 가지고있지않음
  • 2) img 태그 : 처음부터 이미지크기만큼 너비와 높이를 가지고있음
  • 3) select 태그: 처음부터 선택이 가능하도록 공간을 만들어 줌(작은 사각형의 모양을 가짐)
  • 4) input 태그 : 처음부터 글자를 입력받을 수 있도록 작은 사각형의 모양을 가지고있음
