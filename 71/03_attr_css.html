css VS attr

문제1) script 영역에서 문제풀기

css와 attr
css는 꾸며주거나 편집할때 사용하는 작업 (미디어쿼리(해상도별로작업))
css는 속성에 접근하여 바꾸는것이 불가능하다
attr은 꾸며주는 작업이 아니라 해당하는 태그의 속성에 접근하는 작업

아래 내용 매우 중요(attr 활용하기)

1)$("대상").attr("접근할속성","바꿔야할값 입력");

2)$("대상").attr("속성에만 접근");
