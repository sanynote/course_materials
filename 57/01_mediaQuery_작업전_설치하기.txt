1) 구글에서 "크롬 확장 프로그램" 검색
2) "Chrome 웹 스토어" 검색 확인 (접속하기)

3) 아래의 값, 왼쪽 상단의 스토어 검색부분에서 검색하기

    : window resizer 검색
    : 제공업체 : coolx10.com

4) 가장 상단의 검색된 항목 클릭 후, 파란색 버튼 "크롬에 추가" 클릭

    : 확장 프로그램 추가

5) 크롬 브라우저 왼쪽 상단에 추가된 것 확인

    tip) 만약에 설치된것을 확인하기 어렵다면, 오른쪽 상단의 퍼즐모양 아이콘 클릭

    : 그 안에 들어가서 압정 모양 아이콘 누르고 고정시키기

6) 설치목적
    기본적으로 많은 브라우저들이 500px이하로 줄어들지않음
    : 모바일 기종 중에 320px 사이즈를 표현하기 어려움

7) 오른쪽 상단의 고정되어있는 window resizer 버튼 클릭

    : 초록색 버튼 클릭
    : 다양한 해상도 확인 가능

8) 매우중요)
    팝업창 띄우기

        1) 사이즈들 오른쪽에 세로로 나열된 아이콘 확인하기
        2) 위에서 두번째에 있는 Open as popup 이라는 문구가 뜨는 아이콘 클릭하기

        3) 해당하는 사이즈로 확인도 가능하고, 자유롭게 늘리고 줄일수가 있음
